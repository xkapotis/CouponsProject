2.3.2 Transform the expiration values in hours

Suppose your dataframe is called df and your column is ‘expiration’ Function to convert any single value to hours

Code 
convert_to_hours <- function(x) {
  # Extract all number-unit pairs (works for compound like "1d 2h 30m")
  matches <- str_match_all(x, "(\\d+\\.?\\d*)\\s*([a-zA-Z]+)")[[1]]
  
  if(nrow(matches) == 0) return(NA_real_)  # fallback if no match
  total_hours <- 0
  
  for(i in 1:nrow(matches)) {
    num <- as.numeric(matches[i,2])
    unit <- tolower(matches[i,3])
    
    total_hours <- total_hours + switch(unit,
                                        "m" = num / 60,
                                        "h" = num,
                                        "d" = num * 24,
                                        "w" = num * 24 * 7,
                                        NA_real_)
  }
  
  return(total_hours)
}

6 Marketing Campaign Strategy

Based on the modeling results:

  • Random Forest achieved the highest AUC and strongest predictive power.
  • High-frequency coffee visitors show highest responsiveness.
  • Time-of-day effects suggest targeted push timing.
  • Coupon type performance varies significantly.

6.0.1 Proposed Campaign:

  1. Target high-frequency coffee consumers.
  2. Schedule promotions during high-acceptance time windows.
  3. Allocate more budget to top-performing coupon categories.
  4. Use predictive scores to send personalized offers.

### Transform the expiration values in hours

Suppose your dataframe is called df and your column is 'expiration'
Function to convert any single value to hours

```{R}
convert_to_hours <- function(x) {
  # Extract all number-unit pairs (works for compound like "1d 2h 30m")
  matches <- str_match_all(x, "(\\d+\\.?\\d*)\\s*([a-zA-Z]+)")[[1]]
  
  if(nrow(matches) == 0) return(NA_real_)  # fallback if no match
  total_hours <- 0
  
  for(i in 1:nrow(matches)) {
    num <- as.numeric(matches[i,2])
    unit <- tolower(matches[i,3])
    
    total_hours <- total_hours + switch(unit,
                                        "m" = num / 60,
                                        "h" = num,
                                        "d" = num * 24,
                                        "w" = num * 24 * 7,
                                        NA_real_)
  }
  
  return(total_hours)
}
```

Apply function to your whole column dynamically

Check result

# Marketing Campaign Strategy

Based on the modeling results:

-   Random Forest achieved the highest AUC and strongest predictive power.
-   High-frequency coffee visitors show highest responsiveness.
-   Time-of-day effects suggest targeted push timing.
-   Coupon type performance varies significantly.

### Proposed Campaign:

1.  Target high-frequency coffee consumers.
2.  Schedule promotions during high-acceptance time windows.
3.  Allocate more budget to top-performing coupon categories.
4.  Use predictive scores to send personalized offers.
